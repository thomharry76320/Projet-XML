Help

URL Méthode Résumer de l'opération
/ GET Affiche la page d'acceuil de notre application
Format de retour : HTML
/help GET Affiche la page contenant les informations d’aide qui contient la liste des opérations de l'application
Format de retour : HTML
/cv24/resume/xml GET Affiche la liste des CV stockés de manières résumé
Format de retour : XML
/cv24/resume GET Affiche la liste des CV stockés de manières résumé
Format de retour : HTML
/cv24/xml?id=<id> GET Affiche le contenu complet du CV dont l’identifiant est transmis en paramètre ou une erreur en cas d'échec de l'opération.
Format attendue : L'id est un nombre
Format de retour : XML
/cv24/html?id=<id> GET Affiche le contenu complet du CV dont l’identifiant est transmis en paramètre ou une erreur en cas d'échec de l'opération.
Format attendue : L'id est un nombre
Format de retour : HTML
cv24/insert POST Ajoute un CV dans la base de données s'il est valide, sinon renvoie une erreur.
Format attendue : Flux XML respectant la spécification cv24 à ajouter, conforme au schéma xsd.
Format de retour : XML avec l'id du CV ajouté et un message indiquant qu'il est insérer si 'lopération réussis ou sinon un message d'erreur
/cv24/delete?id=<id> DELETE Supprime un CV dont l’identifiant est transmis en paramètre ou une erreur en cas d'échec de l'opération.
Format attendue : L'id est un nombre
Format de retour : XML avec l'id du CV supprimé et un message indiquant qu'il a été supprimé si 'lopération réussis ou sinon un message d'erreur
